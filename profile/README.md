# Welcome to Temporal Community!
This is a spin-off of https://github.com/temporalio/ for Community-focused initiatives and/or experimental code.

Come say hi in our [Temporal Community Slack](https://t.mp/slack)!

## Present About Temporal in Your Local Community with Meetup in a Box!

Do you want to present about Temporal to your local community, but don't have the time to create an entire presentation?
No worries!
The Developer Relations team is proud to announce their latest initiative, Meetup in a Box!

[Check out our Meetup in a Box Program](https://learn.temporal.io/meetup_in_a_box)

## Submit Temporal code to our Code Exchange!

Have you built something cool with Temporal showing off an example use case? Or maybe a helper application that improves
Temporal's developer expreience?

Enter our [Temporal Code Exchange](https://temporal.io/code-exchange), which features both Temporal-created and
Community-created open source projects to help folks hit the ground running with Temporal!

[Submit your project](https://github.com/temporal-community/code-exchange?tab=readme-ov-file#submit-a-code-exchange-project)
and/or look for [suggested projects](https://github.com/temporal-community/code-exchange/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22code%20exchange%20suggestion%22)
to work on!
